Locating genomic regions that are "just right".

* Documentation: http://goldilocks.readthedocs.org.

Requirements
------------
To use;

* numpy

To test;

* tox
* pytest

For coverage;

* nose
* python-coveralls

Installation
------------

::

    $ pip install goldilocks
